| Name                            | Default |
|---------------------------------|---------|
| SABNZBD__PORT                   |         |
| SABNZBD__API_KEY                |         |
| SABNZBD__NZB_KEY                |         |
| SABNZBD__HOST_WHITELIST_ENTRIES |         |
